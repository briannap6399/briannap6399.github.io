SEP 11 Freedom Project

Over the course of SEP11, which was based on an introduction to a beginner friendly programming language named 'Javascript', students were tasked with creating a final project that reflected what they had learned through the base course, as well as through websites such as Kaboom. This project had no limits, and it could truly be anything (as long as it was school appropriate of course). Since we had a lot more creative freedom in this project compared to the last, I wanted to make a game that would not only help students prepare for their Spanish Regents, but also help anyone who wants to start learning Spanish in general. With the help of my partner, Alberto Gonzalez Jr, we both individually split up the core mechanics of the project and, even after all the struggle we've faced, ended with a project we genuinely feel super proud of!
